Core Features

Multiple Minion Types

Includes Miners (ores, blocks, wood), Farmers (crops), and Foragers (plants).

Configurable Targets

Set the specific block or crop a minion works on directly via its GUI (Slot 9 by default).

Customizable GUI

Design the minion's inventory layout using a config file (YAML based on JSON designer).

Fuel System

Optionally require fuel for minions to operate, adding depth to automation.

Storage Options

Minions feature internal storage and can link to adjacent chests.

Ownership & Locking

Player-owned minions with options for co-op access and locking.

Upgrades & Enhancements

Includes configurable upgrade costs, minion linking, auto-smelting, and auto-compacting.

Armor Stand Visuals

Minions are represented by customizable Armor Stands.

YAML Configuration

Easy-to-manage configuration files for settings and GUI layout.

Getting Started

1. Obtain Minion Item: Craft or obtain the specific minion placer item (details defined in config).

2. Place Minion: Right-click on a suitable block with the placer item.

3. Configure: Right-click the placed minion (Armor Stand) to open its GUI.

4. Set Target: Place the block or crop you want the minion to work with into the designated configuration slot (Slot 9 in your example layout).

5. Add Fuel (Optional): If required, add fuel to the designated fuel slot (Slot 18 in your example layout).

6. Manage: Use GUI options for upgrades, collection, linking chests, and pickup (Head icon in Slot 0 in your example).

7. Commands: Use commands like `/minion list` and `/minion collect` (permissions may apply).
